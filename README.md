## 📖 關於實體合本：故事另有結局

玩完測驗了嗎？歡迎來看看我們的實體書：
當童話遇上 BL，加上 30 個元素標籤，王子與王子的幸福結局將被徹底改寫。

這裡有純愛，也有虐戀；
有純潔清水，更多的是深夜福利。
儘管風格百變，這 10 個故事都是不折不扣的 **暗黑童話**。

*準備好顛覆你對童話的想像了嗎？*
*來，走進充滿瘋狂與誘惑的黑森林⋯⋯*
